# 🥩 La Estancia - Parrilla Premium (Restaurant Page)

Una Single Page Application (SPA) que simula la web de una parrilla argentina de alta categoría. Este proyecto fue construido para poner en práctica la manipulación dinámica del DOM usando JavaScript puro y la configuración de entornos de desarrollo con Webpack, como parte del currículum de [The Odin Project](https://www.theodinproject.com/).

🔗 **[¡Mira el proyecto en vivo aquí!](https://Tomas2439.github.io/restaurant-menu/)**

## ✨ Características
- **Navegación SPA:** Cambio entre pestañas (Inicio, Menú, Contacto) de forma instantánea sin recargar la página HTML.
- **Manipulación del DOM:** Todo el contenido, incluyendo imágenes y textos, se genera e inyecta dinámicamente usando JavaScript (`document.createElement`).
- **Arquitectura Modular:** Código dividido en módulos ES6 (`home.js`, `menu.js`, `contact.js`) para una mejor mantenibilidad.
- **Diseño Premium:** Interfaz sobria y elegante enfocada en tipografías serif y colores cálidos/oscuros.
- **Bundling:** Empaquetado y optimización de assets (JS, CSS, Imágenes) automatizado con Webpack.

## 🛠️ Tecnologías Utilizadas
- HTML5
- CSS3 (Flexbox, animaciones, variables nativas)
- Vanilla JavaScript (ES6 Modules)
- Webpack (con `style-loader`, `css-loader` y `html-webpack-plugin`)
- pnpm (Gestor de paquetes)
